Download the Web.

Download audio & video resources, scrape various content types, direct download url paths and retrieve torrent file contents quickly and easily.

Downloading resources from the web should be as quick and easy as possible.

What do you want to download?

Enter the URL or Torrent URI of the resource you want to download. Web DL's handlers will show if they can be used. Simply pick a handler to decide how to download and process your request.

How do you want to download it?

After selecting a handler you can configure the handler specifics (when available) to further specify how to download and process the request.

It will be handled.

Your request will be queued, and when resources are available, will be downloaded and processed. You'll get live updates as your request starts downloading and processing.

Get your results.

After Web DL has finished downloading and processing your request you can view it's details, download the results and go deep into the raw data and logs. There is even a timeline!

Features

Here are some select features Web DL offers to make it even more safer and easier to use.

User management
Create and manage different authenticated users to separate request securely and manage access to them.
Secure
All requests are created locally on your own devices. Only you can see your requests and it's contents.
Storage
Requests are stored in your own cloud (on the device Web DL runs on). Client devices that interact with Web DL only receive the files you decide to download.
Lightweight
Web DL runs light! It's first iteration was created for usage on a Raspberry Pi 3 B+. The latest version of Web DL only requires 2GB of memory and can be run on any ARM device. Raspberry Pi 4 Model B anyone?
Many supported sites
Web DL supports many different web resources. Download from YouTube, Soundcloud, Mixcloud, Imgur, torrents and many others.
Even more features
From compressing all downloaded resource files to live status updates using a secure websocket connection, Web DL contains even more various features to make it easier to use.

How to install

Setting up Web DL requires some manual steps, but the setup process has been designed and developed to be as easy as possible. All you need is git and Docker.

Clone

Start by cloning the API and Website repositories from the GitHub page linked below.

Configure

Copy the following files and input the empty fields. 

.env.dist
-> 
.env

Execute

Install Docker and Docker compose in order to be able to build and run the required API and website containers.

Deployment

Run the following command to build and start it all. 

make build && make start
